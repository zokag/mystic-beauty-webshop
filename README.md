# Mystic Beauty – Web Shop Project

Mystic Beauty is a frontend web shop project built using JavaScript and JSON.

## Features

- Product listing from JSON data
- Search functionality
- Filtering by category and subcategory
- Sorting products (price, rating, etc.)
- Shopping cart system using localStorage
- Responsive design

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- JSON

## Description

This project simulates a real e-commerce website where users can browse products, filter and search items, and add them to the shopping cart.

The focus of the project is on dynamic content rendering, user interaction and clean UI design.
